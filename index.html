Estructura y Operación de un Framework JS

Los Frameworks JavaScript como React, Angular y Vue.js proporcionan una estructura organizada para el desarrollo FrontEnd. Facilitan la creación de aplicaciones con una arquitectura clara y separada en capas. Utilizan el concepto de Client-Side Rendering, donde la mayor parte del procesamiento se realiza en el navegador, mejorando la experiencia del usuario.

Gestión de Dependencias y Herramientas de Creación

Los Frameworks JS usan herramientas como NPM (Node Package Manager) o Yarn para gestionar dependencias externas, es decir, bibliotecas y módulos de terceros que facilitan el desarrollo. Estas herramientas permiten instalar y actualizar paquetes necesarios para tu proyecto.

Además, los frameworks como React utilizan herramientas de creación de proyectos como create-react-app, que generan un entorno de desarrollo listo para comenzar a trabajar.

Desarrollo Basado en Componentes – JSX / PROPS / CHILDREN

Los componentes son la base del desarrollo con frameworks JS. Estos permiten dividir la interfaz de usuario en partes reutilizables. Un componente encapsula la lógica, los estilos y la vista en un único lugar. JSX (JavaScript XML) es una extensión de sintaxis para JavaScript utilizada en React para escribir HTML dentro del código JS.

  • PROPS: Son propiedades que permiten pasar datos de un componente padre a un componente hijo.
  • CHILDREN: Los elementos children representan el contenido que puede ser inyectado en un componente.

Utilización de Estilos en Frameworks JS

Integrar estilos en una aplicación creada con un framework JavaScript puede hacerse de varias maneras:

  • Styles Inline: Permite definir estilos directamente en los elementos HTML utilizando la propiedad style en los componentes.
  • Style Sheets: Se utilizan archivos CSS separados para mantener los estilos de forma organizada. Los estilos se aplican a los componentes a través de clases.
  • Styles Modules: En React y otros frameworks, los módulos de estilos permiten importar archivos CSS directamente en los componentes, garantizando un ámbito local para los estilos.
  • Styled Components: Es una biblioteca que permite escribir CSS dentro del archivo de JavaScript, generando componentes estilizados dinámicamente.
  • Framework CSS: Los frameworks CSS como Bootstrap o Tailwind se integran fácilmente en los proyectos JavaScript, facilitando la creación de diseños responsivos.
